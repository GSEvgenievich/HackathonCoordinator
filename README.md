# Hackathon Coordinator 🚀

**Подсистема для координации деятельности команд на хакатоне** — десктопное приложение для организаторов, капитанов и участников соревнований по программированию. Позволяет управлять соревнованиями, командами, задачами, обеспечивать коммуникацию и интеграцию с GitHub.

## Особенности

- 🏆 **Управление соревнованиями** — создание, редактирование, мониторинг хакатонов
- 👥 **Работа с командами** — система приглашений, распределение ролей, назначение капитанов
- 📋 **Управление задачами** — создание, назначение, отслеживание прогресса
- 💬 **Встроенная коммуникация** — командные чаты и чаты задач в реальном времени
- 🔗 **Интеграция с GitHub** — автоматическое создание репозиториев и веток
- 🔔 **Умные уведомления** — оповещения о событиях в системе
- 🎨 **Кастомизация интерфейса** — 6 цветовых тем (светлая, темная, сезонные)
- 📊 **Экспорт данных** — выгрузка информации в Excel
- 🔐 **Безопасность** — JWT-аутентификация, шифрование данных, разграничение прав

## Технологический стек

### Backend:
- **Язык:** C# (.NET 8)
- **Фреймворк:** ASP.NET Core Web API
- **ORM:** Entity Framework Core
- **Аутентификация:** JWT (JSON Web Tokens)
- **Реальное время:** SignalR
- **Шифрование:** AES-256

### Frontend (клиент):
- **Платформа:** WPF (.NET 8)
- **Паттерн:** MVVM (Model-View-ViewModel)
- **Стилизация:** XAML, динамические темы
- **HTTP-клиент:** HttpClient

### База данных:
- **СУБД:** MySQL 8.0
- **Миграции:** EF Core Migrations

### Инфраструктура:
- **Контейнеризация:** Docker, Docker Compose
- **Администрирование БД:** phpMyAdmin

## Установка и запуск

### Предварительные требования

**Для разработки:**
- .NET 8 SDK
- Visual Studio 2022+ или Rider
- MySQL 8.x
- Docker Desktop (опционально)

**Для запуска готового приложения:**
- .NET 8 Desktop Runtime
- Windows 10 (1809+) или Windows 11
- Стабильное интернет-соединение

### Локальная разработка

#### 1. Клонирование репозитория
```bash
git clone https://github.com/GSEvgenievich/HackathonCoordinator
cd HackathonCoordinator
```

#### 2. Настройка базы данных
```bash
# Создание базы данных
mysql -u root -p
CREATE DATABASE HackathonCoordinatorDb;
```

#### 3. Запуск Backend API
```bash
cd HackathonCoordinator.WebAPI
dotnet restore
dotnet ef database update
dotnet run
```
API будет доступен по адресу: `http://localhost:5046`

#### 4. Запуск WPF клиента
- Откройте `HackathonCoordinator.sln` в Visual Studio
- Установите стартовым проектом `HackathonCoordinator.WPFClient`
- Нажмите F5 для запуска

### Развертывание через Docker

#### 1. Создание файла конфигурации
```bash
cp .env.example .env
# Отредактируйте .env при необходимости
```

#### 2. Запуск контейнеров
```bash
docker compose up -d
```

#### 3. Проверка работы
- **API:** http://localhost:5046
- **phpMyAdmin:** http://localhost:8081
- **Клиент:** Запустите собранное WPF-приложение

## Конфигурация

### Файл `.env` (обязательные параметры):

```env
# ============================================
# НАСТРОЙКИ БАЗЫ ДАННЫХ (MySQL)
# ============================================

# Пароль для root-пользователя MySQL
DB_ROOT_PASSWORD=RootPassword123!

# Имя базы данных, которая будет создана
DB_NAME=HackathonCoordinatorDb

# Имя пользователя для подключения к БД (не root)
DB_USER=hackathon_user

# Пароль пользователя БД
DB_PASSWORD=UserPassword123!


# ============================================
# НАСТРОЙКИ API
# ============================================

# Среда выполнения (Development или Production)
ASPNETCORE_ENVIRONMENT=Production


# ============================================
# НАСТРОЙКИ JWT АУТЕНТИФИКАЦИИ
# ============================================

# Секретный ключ для подписи JWT-токенов (должен быть сложным)
JWT_KEY=hackathon_c00rd_key_super_194_key_inator_13610

# Издатель токена (обычно домен приложения)
JWT_ISSUER=HackathonCoordinator

# Получатель токена (обычно домен приложения)
JWT_AUDIENCE=HackathonUsers

# Время жизни токена в минутах (2 часа)
JWT_EXPIRE_MINUTES=120


# ============================================
# НАСТРОЙКИ GitHub OAuth
# ============================================

# Client ID из зарегистрированного OAuth приложения GitHub
GITHUB_CLIENT_ID=YOUR_GITHUB_CLIENT_ID

# Client Secret из зарегистрированного OAuth приложения GitHub
GITHUB_CLIENT_SECRET=YOUR_GITHUB_CLIENT_SECRET

# URL для перенаправления после авторизации в GitHub
GITHUB_REDIRECT_URI=http://YOUR_DOMEN_NAME/api/oauth/github-callback


# ============================================
# НАСТРОЙКИ ШИФРОВАНИЯ
# ============================================

# Ключ для AES-256 шифрования GitHub-токенов
# ⚠️ ВАЖНО: Не менять после создания базы данных!
ENCRYPTION_KEY=1my1-02ro1-irdnaa-56dj8y5td6-key-32-10fa2


# ============================================
# НАСТРОЙКИ MINIO (Файловое хранилище)
# ============================================

# Логин администратора MinIO
MINIO_ROOT_USER=minioadmin

# Пароль администратора MinIO
MINIO_ROOT_PASSWORD=minioadmin

# Внутренний адрес MinIO в Docker-сети
MINIO_ENDPOINT=minio:9000

# Публичный адрес MinIO для доступа из WPF-клиента
MINIO_PUBLIC_ENDPOINT=localhost:9000

# Название бакета (контейнера) для файлов
MINIO_BUCKET=hackathon-files

# Использовать SSL (для локальной разработки false)
MINIO_USE_SSL=false
```

## Интеграция с GitHub

### Настройка OAuth приложения:

1. Перейдите в [GitHub Developer Settings](https://github.com/settings/developers)
2. Нажмите "New OAuth App"
3. Заполните поля:
   - **Application name:** Hackathon Coordinator
   - **Homepage URL:** `http://your-domain.com`
   - **Authorization callback URL:** `http://your-domain.com/api/oauth/github-callback`
4. После создания скопируйте `Client ID` и `Client Secret`
5. Вставьте их в файл `.env`
6. Перезапустите контейнеры: `docker compose restart api`

## Документация

### Роли пользователей:

| Роль | Возможности |
|------|-------------|
| **Организатор** | Создание/редактирование соревнований, управление командами, назначение капитанов |
| **Капитан** | Управление задачами команды, распределение заданий, интеграция с GitHub |
| **Участник** | Просмотр задач, участие в чатах, запрос проверки выполненных задач |
| **Свободный** | Поиск команд, присоединение по приглашениям, редактирование профиля |
